# Web Admin Dashboard README

This is the starting point for the Web Admin Dashboard for the My Diary project.

## How to run (for web)

1. Open terminal in this directory.
2. Run:
   flutter run -d chrome -t lib/web_admin/main_web_admin.dart

## Structure
- `web_admin_dashboard.dart`: Main dashboard UI for web.
- `main_web_admin.dart`: Entry point for web admin app.
- `pubspec.yaml`: Project dependencies.

You can now add more widgets and features for your web admin dashboard here.
